A react-native mobile application designed to aid in the viewing of UVA courses. Designed by myself and with Jason Ashley. The main functionalities of the app run through our Nodejs backend, which is hosted on Heroku and uses Mongo Atlas to store course/user data. RealitariumVirtual Reality Planetarium

A virtual planetarium being designed with the intent to be used in a classroom setting at UVA. I am working closely with Dr. Ed Murphy, a Professor of Astronomy also at UVA. The end goal of this project is to have a mobile application which students can use to enter a planetarium, virtually, during lecture using a device similar to Google Cardboard.
