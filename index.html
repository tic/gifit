LASE Lab SiteDatabase Viewer+Editor

Application designed in react-native that extends support to web via the Expo web bundler. Will serve as the replacement website for the researchers at LASE, a graduate/post-grad research lab at UT Austin. In addition to offering the static contents linked previously, the application also serves as the interaction between researchers and their SQL database, allowing for easy display, filtering, and creation of data. BotarinoAdvanced Discord Bot

Written with Node using the discord.js library, Botarino is an advanced chatbot which utilizes many JS features new to ES6. Based on a deeply OO configuration, Botarino is capable of hot-reloads triggered by chat commands, multiple layers of fault tolerance, and connection to MongoDB for logging, analytics, and stat tracking.
